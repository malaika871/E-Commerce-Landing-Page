## Getting Started

### Prerequisites
- Node.js 18+ installed

### Installation

Install dependencies:
```bash
npm install
```

### Running the Development Server

Start the development server:
```bash
npm run dev
```

The application will open at `http://localhost:3001/`

### Building for Production

Create a production build:
```bash
npm run build
```
